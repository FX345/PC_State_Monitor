# PcGuardianLite 下载使用说明

## 给普通用户

1. 打开 GitHub 仓库的 Releases 页面。
2. 下载最新版本里的 `PcGuardianLiteSetup.exe`。
3. 双击 `PcGuardianLiteSetup.exe`。
4. 按需勾选“在桌面创建快捷方式”和“安装完成后立即启动”。
5. 安装完成后，桌面右上角会出现悬浮球。

不要下载 `Source code.zip` 或 `Source code.tar.gz` 来当安装包使用。源码包需要开发环境，普通用户直接打开会用不了。

## 使用方式

- 单击悬浮球：展开或收起状态面板。
- 拖动悬浮球：移动位置。
- 右键悬浮球：切换显示内存、CPU、网速、自动轮播，设置透明度，隐藏到托盘或退出。
- 托盘图标：双击可重新显示悬浮球。
- 网络页：可查看实时上传下载流量，也可以运行 Speed Test。
- 清理页：先点“扫描垃圾”，确认列表后勾选项目，再点“清理选中项”。
- 报告页：生成电脑体检报告、网络报告，或打开报告目录。

## 下载速度和上传速度是什么意思

主面板和悬浮球上的“下载 / 上传”是当前电脑所有活动网卡的实时流量。

- 下载：电脑每秒正在接收多少数据。
- 上传：电脑每秒正在发送多少数据。

它不是宽带套餐的最大速度。要估算宽带速度，请进入“网络”页点击 `Speed Test`。

## 卸载

可以从 Windows 设置卸载：

```text
Settings > Apps > Installed apps > PcGuardianLite > Uninstall
```

也可以打开安装目录运行：

```text
%LOCALAPPDATA%\PcGuardianLite\Uninstall PcGuardianLite.bat
```

卸载会删除程序和快捷方式，但会保留 `ScriptReports` 里的报告文件。

## 开发者打包

在源码目录双击：

```text
build_release.bat
```

生成结果在：

```text
release\PcGuardianLiteSetup.exe
```

把 `release` 文件夹里的安装包发给别人即可。
